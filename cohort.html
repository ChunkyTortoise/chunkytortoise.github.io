FAQ

Do I need a specific LLM provider?

No. All labs run with mock agents and local logic. When you're ready to add real LLM calls (Claude, GPT-4, Gemini), the course shows you exactly where to plug them in. The architecture is provider-agnostic by design.

How much time should I expect per week?

Plan for 6–8 hours: 3 hours of live sessions and 3–5 hours on labs and reading. Labs have stretch goals if you want to go deeper. All sessions are recorded within 24 hours for async viewing.

Is this the same content as the open-source starter kit?

The multi-agent-starter-kit is the foundation. The course adds 4 extended labs, live instruction, written code review, architecture deep-dives, production deployment patterns, and CI integration that go well beyond the README examples.

Do I need to know a specific AI framework (LangChain, CrewAI, etc.)?

No. This course is framework-independent by design. The orchestration layer is pure Python with only Pydantic as a dependency. If you use a framework at work, you'll learn the primitives underneath it — making you better at using the framework too.
